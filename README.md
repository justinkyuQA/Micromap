# MicroMap

MicroMap is a tiny educational network mapping tool.

It resolves a host and checks a small list of common TCP ports.

## Usage

```bash
python3 -m micromap google.com
python3 -m micromap 127.0.0.1

---

## Overview

Educational visualization tool for understanding simple network layouts.

- Lightweight command-line utility
- Cross-platform Python

- Educational examples
